# 🚀 AETHER TAURI & ECOSYSTEM — MARKETING KIT

This folder contains the complete, ready-to-deploy advertising assets for the Aether ecosystem.

## 📁 Directory Structure
```
advertising_kit/
├── COPY_AND_PASTE_POSTS.md  # Social media copy for Twitter, Reddit, LinkedIn
├── ANIMATED_DEMO_AD.html    # Self-contained animated HTML presentation (Video replacement)
└── images/                  # High-quality SVG assets for headers and social media
    ├── hero-banner.svg
    ├── comparison-chart.svg
    ├── ecosystem-graphic.svg
    ├── terminal-screenshot.svg
    └── hero.svg
```

## 🎬 How to Use the Animated Ad (`ANIMATED_DEMO_AD.html`)
Since generating high-quality MP4 files programmatically is not possible without external dependencies, we built a fully self-contained HTML/CSS animated sequence.

1. Double-click `ANIMATED_DEMO_AD.html` to open it in your browser.
2. It plays a 5-scene cinematic sequence demonstrating Aether Tauri (typing effects, system status, etc.).
3. **To get a video:** Use a screen recording tool (like OBS or QuickTime) to record the browser window while the animation plays. Add audio in post-production if desired.

## 📝 How to Use the Copy (`COPY_AND_PASTE_POSTS.md`)
This document isolates the viral hooks and pre-written posts from the `LAUNCH_PACK.md`. It provides exact text formatted for Twitter/X threads, Reddit subreddits (r/LocalLLaMA, r/privacy), and LinkedIn. 

1. Simply copy the text for the relevant day.
2. Attach the suggested image from the `images/` folder.
3. Post.

## 🖼️ How to Use the Images
The `images/` directory contains `.svg` files.
- If a platform (like Twitter/X) does not accept SVGs, you can open them in your browser and take a screenshot, or use an online SVG-to-PNG converter.
- `hero-banner.svg` is best for the main launch post.
- `comparison-chart.svg` is best for the "Privacy Angle" post.
- `ecosystem-graphic.svg` is best for the "Ecosystem/Sideload" post.
