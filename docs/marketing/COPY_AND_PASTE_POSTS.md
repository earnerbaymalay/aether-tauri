**Body:**
Hey HN,

I’ve been increasingly uncomfortable with the centralization of AI tools and the necessity of sending private code and thoughts to cloud providers. To solve this for myself, I built Aether Tauri: a local-first neural workstation that bridges a Tauri/TypeScript frontend with a Python agent core.

It relies on Ollama for inference and leverages the OpenClaw CLI for tool execution (ReAct loops, MCP support) without ever phoning home. 

Some interesting architectural bits:
- **AetherVault:** A background shadow-monitor distills conversations into Markdown fragments (Obsidian compatible) for long-term RAG context.
- **Neural Pathways:** It doesn't rely on a single massive model. You can switch between specialized 3B-8B models (e.g., Qwen-Coder vs. DeepSeek-R1) depending on the task.
- **AetherLink:** A lightweight P2P protocol running on port 8888 to sync memory fragments across local network nodes.
- **Nexus Shield:** Hardening tools to kill OS telemetry and throttle background processes during heavy inference.

The entire stack is 100% offline and MIT licensed. I’d love your feedback on the architecture, the MCP integration approach, or local LLM agent loops in general.
