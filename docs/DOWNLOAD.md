### 🚀 One-Click Installation

Aether is now bundled with its own Engine. You no longer need to install Python or configure complex settings manually.

#### 🍎 macOS
1. Download **`Aether-macOS.dmg`**.
2. Drag **Aether** to your **Applications** folder.
3. **First Launch:** Right-click the app and select "Open" (to bypass the Alpha security check).

#### 🪟 Windows
1. Download **`Aether-Windows.msi`**.
2. Run the installer and follow the prompts.
3. **Launch.** Aether will automatically verify your hardware and stand by.

---

### 🛡️ Pre-requisites
*   **Ollama:** Aether uses Ollama for local model inference. If you don't have it, Aether will prompt you to install it on your first launch.
