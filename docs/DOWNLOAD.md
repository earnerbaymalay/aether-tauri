### 🪟 Windows Installation
1. Download the latest `Aether-Windows.msi` from the [Releases](https://github.com/earnerbaymalay/aether-tauri/releases) page.
2. Run the installer. If Windows SmartScreen appears, click "More Info" -> "Run Anyway" (we are currently in the process of code-signing).
3. **Requirement:** Ensure [Ollama](https://ollama.ai/) is installed and running in your taskbar.

### 🍎 macOS Installation
1. Download `Aether-macOS.dmg`.
2. Open the DMG and drag **Aether** to your **Applications** folder.
3. Right-click the app and select "Open" for the first launch to bypass the unidentified developer warning.
4. **Requirement:** [Ollama](https://ollama.ai/) must be active.

### 🐧 Linux Installation
1. Download the `Aether.AppImage`.
2. Right-click -> Properties -> Permissions -> **Allow executing file as program**.
3. Double-click to launch.
